Was passiert da? Situationen beschreiben

Passiv flüssig nutzen — Bildbeschreibung & Reporter-Sprache

Mini-Wiederholung Passiv
PRÄSENS-PASSIV: werden + Partizip II → "Hier wird gebaut."
Mit Modalverb: muss/soll/darf + P.II + werden → "Das muss gemacht werden."
Redemittel: Situationen einleiten
Bildbeschreibung
"Auf diesem Bild sieht man…"
"Hier wird gerade…"
"Im Vordergrund wird…"
"Im Hintergrund sieht man, wie…"
"Auf der linken/rechten Seite wird…"
Genauer beschreiben
"Während im Vordergrund X gemacht wird, wird im Hintergrund Y…"
"Anscheinend wird hier…"
"Was hier passiert, ist… (+ Passiv)"
🎯 Ziel heute: Mindestens 6 Passiv-Sätze pro Bildbeschreibung — flüssig und lebendig!

🏗️Auf der Baustelle
Was passiert hier? Beschreibe mit 6+ Passiv-Sätzen!
ein Haus bauen
Steine herauftragen
Mauer streichen
Kran bedienen
Material transportieren
hart arbeiten
Auf diesem Bild sieht man eine Baustelle. Im Vordergrund wird ein Haus gebaut. Steine werden von Arbeitern heraufgetragen. Im Hintergrund wird eine Mauer gestrichen. Ein Kran wird bedient, um schwere Materialien zu transportieren. Insgesamt wird hier hart gearbeitet.
🍳In der Küche (Festessen)
Großes Festessen wird vorbereitet — beschreibe!
Suppe kochen
Hähnchen braten
Gemüse schneiden
Salat waschen
Kuchen dekorieren
Teller spülen
Hier wird ein Festessen vorbereitet. Auf dem Herd wird eine Suppe gekocht. Im Ofen wird ein Hähnchen gebraten. Auf dem Tisch werden Gemüse geschnitten und Salat gewaschen. Ein Kuchen wird gerade dekoriert. Im Hintergrund werden Teller und Gläser gespült.
💇Im Friseursalon
Viel los — viele Aktivitäten gleichzeitig!
Haare schneiden
Tönung auftragen
Haare waschen
an Kasse bezahlen
geplaudert
Im Friseursalon werden viele Dinge gleichzeitig gemacht. Eine Kundin bekommt die Haare geschnitten. Einer anderen wird eine Tönung aufgetragen. Im Hintergrund werden Haare gewaschen. An der Kasse wird bezahlt. Es wird viel geplaudert.
🚉Bahnhof zur Hauptverkehrszeit
Reisende, Schaffner, Koffer — alles in Bewegung!
Tickets kontrollieren
Koffer schleppen
Durchsagen machen
Zeitungen verkaufen
Kaffee bestellen
Im Bahnhof werden viele Dinge gleichzeitig erledigt. An den Schaltern werden Tickets verkauft. Im Bahnsteig werden Durchsagen gemacht. Koffer werden geschleppt. Im Café wird Kaffee bestellt. Tickets werden von Schaffnern kontrolliert.

Sprechübungen · Rollenspiele

Lebendig erzählen

🎭 Rollenspiel 1 — Telefonische Bildbeschreibung
Du beschreibst ein Bild oder eine Szene am Telefon. Dein Partner zeichnet es nur aus deinen Worten!
Person A: Beschreibe Szene mit Passiv-Sätzen ("Auf dem Bild wird…")
Person B: Zeichne nur basierend auf der Beschreibung
Vergleicht am Ende: Wie nahe seid ihr am Original?
🎭 Rollenspiel 2 — Reporter vor Ort
Live-Berichterstattung von einem Event (Konzert, Demo, Sportveranstaltung).
Person A — Reporter: Beschreibe das Geschehen LIVE im Passiv. "Hier wird gerade…"
Person B — Studio: Stelle Nachfragen: "Was passiert jetzt? Wer wird interviewt?"
Ziel: Mindestens 10 Passiv-Sätze + lebendige Beschreibung!
🎯 Wortketten-Spiel im Kreis
Jede:r beschreibt EINE Sache, die an einem Ort gemacht wird. Die Kette wird länger!
Beispiel "Schule":
→ "In der Schule wird gelernt."
→ "In der Schule wird gelernt UND es werden Pausen gemacht."
→ "In der Schule wird gelernt, es werden Pausen gemacht UND Aufsätze werden geschrieben."
… immer länger, bis jemand stockt!
