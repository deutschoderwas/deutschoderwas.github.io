B1/B2 · Der Fokus-Shift

Wann nutzen wir das Passiv?

Aktiv = WER macht. Passiv = WAS passiert.

🌊 AKTIV
„Anna schreibt etwas."
Wann? Wenn wir wissen UND betonen wollen, WER handelt!
🌸 PASSIV
„Etwas wird geschrieben."
Wann? Wenn die HANDLUNG wichtiger ist als die Person!
Die goldene Formel
WERDEN + Partizip II = PASSIV
🎯 Ziel heute: Passiv FLÜSSIG nutzen, um Situationen lebendig zu beschreiben — wie ein Reporter vor Ort!
Warm-up: Schaue um dich
🔍 Mini-Detektive: Was passiert gerade um dich herum? Wer macht was — und wer ist „betroffen"? Versuche 3 Sätze: einer im Aktiv, einer im Passiv, einer im BEKOMMEN-Passiv!
Beispiele · Polaroids

Drei Schnappschüsse aus dem Alltag

🏗️
POLAROID 1
„Hier wird gebaut."
Baustellen-Passiv — die AKTION läuft, die Arbeiter sind zweitrangig.
🍲
POLAROID 2
„Die Suppe wird gekocht."
Küchen-Passiv — das OBJEKT (Suppe) rückt in den Vordergrund.
🔧
POLAROID 3
„Das Auto wird repariert."
Werkstatt-Passiv — was MIT dem Auto passiert, ist entscheidend.

Drei komplexe Beschreibungen

Fallstudie 1 · Baustelle
🏗️ Auf der Baustelle
Auf diesem Bild sieht man eine Baustelle. Im Vordergrund wird ein Haus gebaut. Steine werden von Arbeitern heraufgetragen. Im Hintergrund wird eine Mauer gestrichen. Ein Kran wird bedient, um schwere Materialien zu transportieren. Insgesamt wird hier hart gearbeitet.
✓ 5 Passiv-Konstruktionen
Fallstudie 2 · Küche
🍲 In der Küche
Hier wird ein Festessen vorbereitet. Auf dem Herd wird eine Suppe gekocht. Im Ofen wird ein Hähnchen gebraten. Auf dem Tisch werden Gemüse geschnitten und Salat gewaschen. Ein Kuchen wird gerade dekoriert. Im Hintergrund werden Teller und Gläser gespült.
✓ 6 Passiv-Konstruktionen
Fallstudie 3 · Friseursalon
💇‍♀️ Im Friseursalon
Im Friseursalon werden viele Dinge gleichzeitig gemacht. Eine Kundin bekommt die Haare geschnitten. Einer anderen wird eine Tönung aufgetragen. Im Hintergrund werden Haare gewaschen. An der Kasse wird bezahlt. Es wird viel geplaudert.
✓ 6 Passiv-Konstruktionen (1× BEKOMMEN-Passiv!)
🎯 Beobachtet: Im Friseursalon kommt eine NEUE Form vor — das BEKOMMEN-Passiv. Mehr dazu in Phase 3!
B2 · Achtung Spezial

Das BEKOMMEN-Passiv

Alternative für Passiv mit „lassen"

Die Formel
Person (Empfänger) + BEKOMMEN + Objekt + PARTIZIP II
Bekommen-Passiv
„Sie bekommt die Haare geschnitten."
=
Werden-Passiv
„Ihr werden die Haare geschnitten."
Mehr Beispiele aus dem Alltag
Beim Friseur
„Ich bekomme die Haare gefärbt."
Beim Arzt
„Er bekommt einen Verband angelegt."
In der Werkstatt
„Sie bekommt ihr Auto repariert."
Zum Geburtstag
„Er bekommt ein Geschenk geschickt."
⚡ Wichtig: BEKOMMEN-Passiv geht NUR, wenn jemand etwas EMPFÄNGT (Person bekommt etwas gemacht). „Hier bekommt gebaut" ❌ — funktioniert nicht!
Übung · Transformation

Werkzeugkasten

Redemittel für die Bildbeschreibung

B1 — Situationen einleiten
Räumlich
„Auf der linken Seite WIRD…"
Aktion läuft
„Es WIRD GERADE… (gekocht/gebaut/repariert)."
Allgemein
„Auf diesem Bild SIEHT MAN…"
Vordergrund
„Im Vordergrund STEHT eine Person, die…"
Hintergrund
„Im Hintergrund SIEHT MAN, wie…"
Hier & jetzt
„Hier WIRD GERADE…"
B2 — Genauer beschreiben
Gleichzeitigkeit
„Während im Vordergrund X gemacht wird, wird im Hintergrund Y…"
Vermutung
„Anscheinend wird hier… / Es macht den Eindruck, dass…"
Aktiv + Passiv
„Die Person scheint gerade zu… (Aktiv), während sie selbst… (Passiv)."
Zusammenfassung
„Was hier passiert, ist…"
💎 B2-Tipp: Kombiniere mehrere Redemittel für eine fließende, natürliche Beschreibung!

Live-Einsatz: 2 Rollenspiele
📞 Rollenspiel 1: Telefonische Bildbeschreibung
Person A (Beschreiber): Beschreibt ein Bild am Telefon — so viele Passiv-Konstruktionen wie möglich!
Person B (Zeichner): Zeichnet die Szene NUR basierend auf As Beschreibung. Kein Original sehen!
🎯 Am Ende: Vergleichen — wie genau ist das Bild geworden?
📺 Rollenspiel 2: Reporter vor Ort
Szenario: Live-Event (Konzert, Demo, Festival).
Person A (Reporter): Beschreibe LIVE. Was wird gemacht? Mindestens 10 Passiv-Sätze!
Person B (Studio): Stelle Nachfragen — „Was wird gerade vorbereitet?"
🎮 Die Kettenreaktion — das große Sprechspiel
Beschreibe EINE Sache, die gemacht wird — und wiederhole alles Vorherige!
Runde 1: „In der Schule wird gelernt."
Runde 2: „…wird gelernt UND es werden Pausen gemacht."
Runde 3: „…Pausen gemacht UND Aufsätze werden geschrieben."
Runde 4+: Immer länger!
⚠️ Wer einen Fehler macht, scheidet aus!
