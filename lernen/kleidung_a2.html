👔 Im Geschäft
  • Ich suche eine/einen/ein ___ in Größe ___. = I'm looking for a ___ in size ___. –
  • Kann ich das anprobieren? = Can I try that on? –
  • Das steht dir gut! = That suits you well! – Kompliment!
  • Das ist leider ausverkauft. = That's unfortunately sold out. –
🌈 Farben beschreiben
  • Das ist dunkelblau / hellgrün / knallrot. = That is dark blue / light green / bright red. –
  • Das Hemd ist gestreift / kariert / einfarbig. = The shirt is striped / checkered / plain. –
